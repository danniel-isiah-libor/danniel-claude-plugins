# Vue / shadcn-vue / Tailwind

- Vue 3 + `<script setup>`; props typed; composables for shared logic; keep components focused.
- shadcn-vue is built on reka-ui (not Radix/React) — follow reka component APIs.
- Tailwind: utility-first; centralize design tokens; respect the project's Tailwind version (v4 theming differs from v3).
- Type-check is the gate in many Vite setups (`vue-tsc`); unused locals/imports may be hard errors — keep imports clean.
